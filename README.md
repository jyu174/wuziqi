# 五子棋 Gomoku

HTML5/JS 五子棋，19×19 棋盘。支持：

- **人机对战** — 三种难度：简单 / 普通 / 困难（可选执黑或执白）
- **联机对战** — 注册登录后进入大厅，向在线玩家发起挑战，两台电脑各自用浏览器实时对弈
- **用户系统** — 简单的用户名/密码注册与登录

## 运行 Run

```bash
npm install
npm start          # 默认端口 8080，或 PORT=3000 npm start
```

然后浏览器打开 `http://<服务器IP>:8080`（本机为 http://localhost:8080）。
联机对战时，两位玩家访问同一个服务器地址，各自登录后进入大厅互相挑战。

## 规则

自由五子棋（无禁手）：黑先，横/竖/斜任意方向连成五子（或以上）即胜，棋盘下满为平局。

## 技术

- 前端：原生 HTML5 + Canvas + vanilla JS，无框架、无构建步骤（`public/`）
- 后端：Node.js，`ws` WebSocket（唯一依赖）；服务器权威校验落子与胜负（`server.js`）
- AI：模式打分 + 候选点裁剪；困难档为 α-β 剪枝负极大值搜索（深度 6）（`public/js/ai.js`）
- 账号：`crypto.scrypt` 加盐哈希，存于 `users.json`（已 git-ignore）；内存会话 token（`auth.js`）

> 注：认证为业余场景设计（HTTP 明文、内存会话），适合局域网/朋友间使用，未做公网加固。
